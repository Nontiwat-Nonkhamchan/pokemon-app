# Features

- แสดงรายชื่อ Pokémon ทั้งหมด 1351 ตัว
- โหลดข้อมูลแบบ Load More (Pagination)
- แสดงรายละเอียด Pokémon
  - รูปภาพ
  - ชื่อ
  - ประเภท (Type)
  - Base Stats
  - Evolution Chain
  - Pokémon Cry
- ใช้ Material UI
- ใช้ React Skeleton ระหว่างโหลดข้อมูล
- Responsive Design
- หน้า About This Project
